Show Success Message onSubmit

Add this HTML to your website:

<form method="POST" action="https://www.formrocket.me/api/forms/<%=req.query.userId%>/<%=encodeURIComponent(req.query.formName)%>/<%=req.query.formId%>/post/showSuccessMessageOnDone"><input name="example" placeholder="example input" /></form>

Optionally you can add the ?message= parameter to the action URL to customize the success message.
